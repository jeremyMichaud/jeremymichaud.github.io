Detailed Description

Time mananger module (uses RTC).

The time manager will use a RTC to keep the real time information. Once initialized, it will give the count of the real time.

◆ rtc_handler()

static void rtc_handler ( nrf_drv_rtc_int_type_t  int_type)
static

Function to handle rtc overflow and timeout.

The RTC should never overflow. If it reachs the TIMEOUT value, the current value will be added to the base_time and the RTC will restart from 0.
