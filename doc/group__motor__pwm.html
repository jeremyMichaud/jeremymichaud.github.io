Detailed Description

Manage door opening pins by either PWM or GPIO (high or low).

If PWM_ENABLED is set, it will output PWM (10%) on the open pin when the function motor_open() is called and on the close pin when the function motor_close() is called.

If PWM_ENAMBLE is not set. motor_open() will set the open pin to 1 and motor_close() will set it to 0.
