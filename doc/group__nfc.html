Detailed Description

NFC module, will use adafruit_pn532 library to read tags.

This file will read nfc tags, and accept only Type 2 (works usually with NTAG213). It will read the tag and verify with hash (SHA256) and signature (AES128) if the tag is correct.

◆ tag_detect_and_read()

ret_code_t tag_detect_and_read ( void  )

Function for detecting a Tag, identifying its Type and reading data from it.

This function waits for a Tag to appear in the field. When a Tag is detected, Tag Platform Type 2 is identified and appropriate read procedure is run. After reading the data, it will look if the data is correct to open the door (base on id and timestamp).
