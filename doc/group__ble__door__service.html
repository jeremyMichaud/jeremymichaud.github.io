Detailed Description

File to manage Door service.

This file will create the Door service and its characteristics. It will run a callback function when it receive a write event.

Function to send back a byte value (like a return code).

Function for updating and sending new characteristic values.

When this function is called, it will send an update to the connected device with the value given in parameter. This function is supposed to work as a return code in a C stype. For example returning 0 would mean everything is OK (the door has opened) and 1 if an error occured.

Parameters
[in]p_door_serviceDoor Service structure.
[in]door_valueValue to send to the device.
