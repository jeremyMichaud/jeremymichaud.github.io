Detailed Description

File to manage Admin service.

This file will create the Admin service and its characteristics. It will run a callback function when it receive a write event.

uint8_t  status_value 
)

Function to send back a byte value (like a return code).

When this function is called, it will send an update to the connected device with the value given in parameter. This function is supposed to work as a return code in a C stype. For example returning 0 would mean everything is OK and 1 if an error occured.

Parameters
[in]p_adm_serviceAdmin Service structure.
[in]status_valueValue to send to the device.
